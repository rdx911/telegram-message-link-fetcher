Telethon User Guide

Introduction

This script uses the Telethon library to fetch message links from a specified Telegram channel.

Prerequisites

  • Python 3.7+ installed
  • Install Telethon: pip install telethon
  • API ID & API Hash from my.telegram.org

Configuration

SESSION_NAME = "session_name"
api_id = 1234567  # Replace with your API ID
api_hash = "74839201563728495012643895720163"  # Replace with your API Hash

How to Run the Script

  1. Run: python script.py
  2. Enter the Channel ID (You can extract it from the channel URL, e.g., https://web.telegram.org/a/#-1002593905638 gives -1002593905638)
  3. Choose mode: all or range (Enter start and end message ID if selecting range, extracted from message link e.g., https://t.me/c/2593905638/99 where message ID is 99)

Data Source

The script fetches messages from Telegram channels using the Telethon API.

Output

Extracted message links are saved in:

channel_.txt

Error Handling

Error Description
FloodWaitError Waits before retrying
ChatAdminRequiredError Bot lacks admin rights
ChannelPrivateError Cannot access private channel
SessionPasswordNeededError Two-step verification enabled

Conclusion

This script simplifies Telegram message link extraction. For more details, visit Telethon Documentation.
